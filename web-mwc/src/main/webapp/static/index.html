Minimal Spring MVC project with examples

This page contains set of links and forms to demonstrate configuration and possibilities of Spring MVC.

Accessing context root "host"/web-mvc/ will redirect to this index.html page. Used is ViewController functionality, see MvcConfig.configureViewResolvers().

Sample page with controller processing

Calls controller MyBasicController.returnSampleView with mapping /sample Configuration of ViewResolver for JSP pages is in ProgrammaticConfiguration.internalResourceViewResolver(). We override default bean internalResourceViewResolver. If could be used WebMvcConfigurer.configureViewResolvers(ViewResolverRegistry) I don't know. Jsp location is in WEB-INF/view/samplePage.jsp.
